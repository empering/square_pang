# Install & Development

```bash
git clone https://github.com/empering/square_pang.git
cd square_pang

# Install dependencies
npm install

# start server
npm run start

# Client url: http://localhost:4200
```

# Build & Production

```bash
npm run build

## Deploy dist folder to app server
Structure of dist folder:

/dist/server <-- expressjs
/dist/client <-- angular2
```
